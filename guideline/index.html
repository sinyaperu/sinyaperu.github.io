  • 作品全般について

    二次創作・FA は基本的に歓迎です。
    #深夜の抽斗 や、ゲームタイトルのハッシュタグを
    つけて投稿してくださるととても嬉しいです！
    ※以下、二次創作・FA を一律「FA」と表現します。

    許可事項

    FAの作成 / SNSへの公開
    作品の保存 / 個人使用

    禁止事項

    R18 / R18G のFA
    作品の無断転載 / 無断使用 / 自作発言 / AI学習
    作品のイメージを損なう内容の投稿 / 拡散
    深夜ぺるやその関係者、キャラへの誹謗中傷
    作品及びそのFAで金銭を得ること
    その他不適切な行為

    判断に迷うときは
    wavebox にて質問をお願いします

  • ゲームの実況/配信

    条件付きで許可いたします。

    条件

    動画/配信タイトルにゲームタイトルを含めること
    概要欄にゲーム配信ページのURLを記載すること
    禁止事項に抵触しないこと

    禁止事項

    メンバーシップ限定などの有料配信 / 動画
    実況やテキスト無しのプレイ配信 / 動画
